## 1. 页面概述 (Overview)

本页面是策略的 **“信任验证中心”** 与 **“付费转化陷阱”**。

* **双重使命：**
1. **对已完结交易 (下半部)：** 全透明公开，精准到秒，用于验证历史战绩，建立“法医级”信任。
2. **对进行中交易 (上半部)：** 实施“猫鼠游戏”。公开行业与浮盈，隐藏代码与价格，利用 FOMO (错失恐惧) 诱导用户付费解锁。



---

## 2. 页面布局架构 (Layout)

页面布局分为上下两部分：

* **Zone A (Global Header):** 策略身份与实时信号控制台（继承自框架，不做详细展开，但需提及右侧转化区）。
* **Zone B (Transaction List):** 核心交易数据列表。包含 **工具栏**、**动态信号横幅**、**混合状态数据表**。

---

## 3. 核心功能区详解 (Detailed Specs)

### 3.1 顶部信号控制台 (Zone A - Right)

*参考 UI 右上角蓝色区域。*

虽然属于全局 Header，但在本页作用至关重要。

* **左侧状态：** `🟢 实时信号直推` (带呼吸灯) | `支持 微信/短信/App`。
* **右侧按钮：** `⚡️ 立即解锁信号`。
* **联动逻辑：** 点击按钮，弹出权益订阅弹窗。

---

### 3.2 列表工具栏 (Zone B - Toolbar)

位于表格正上方，用于筛选和配置。

* **Display Columns (显示列配置):**
* **默认状态：** 隐藏大部分复选框，保持界面清爽。
* **交互：** 仅保留一个 `⚙️` 齿轮图标在最右侧，点击展开列配置菜单。
* **默认选中列：** `品种`、`交易类型`、`成交价`、`平仓盈亏`。


* **数据更新时间：**
* 显示：`Last Updated: 实时` (针对加密行) / `收盘后` (针对公开行)。



---

### 3.3 动态信号横幅 (Signal Alert Banner) —— *条件触发组件*

*基于“实时信号引导”讨论补充，位于工具栏与表格之间。*

* **触发条件：** 当策略在 **15分钟内** 有新交易产生时显示。
* **UI 样式：** 红色/橙色高亮通栏。
* **文案：** `🔔 刚刚触发买入信号 (10:32)。标的：消费电子。 [点击解锁]`。
* **交互：** 点击直接唤起支付弹窗。

## 4. 交互逻辑 (Interaction Logic)

### 4.1 鼠标悬停 (Hover Effects)

* **加密行：** 当鼠标划过加密行时：
* 整行背景色微亮 (Light highlight)。
* 蓝色 `🔓 解锁` 按钮出现微小的**放大/跳动动画**。
* 鼠标指针变为手型 (`cursor: pointer`)。


* **公开行：** 标准的列表悬停效果。

### 4.2 点击行为 (Click Action)

* **点击加密行 (任意位置)：**
* **动作：** 阻断跳转，弹出 **“权益订阅弹窗 (Subscription Modal)”**。
* **弹窗上下文：** 弹窗标题动态变化为 —— “解锁 [消费电子] 板块的实时代码”。


* **点击公开行：**
* (可选) 跳转至该个股的行情详情页，或无动作。



### 4.3 支付弹窗逻辑 (The Modal)

* *引用之前的讨论：* 弹窗需展示 **免费 / 单策略(推荐) / PRO** 三档权益。
* **默认选中：** 中间的 **“单策略订阅 (¥299/月)”**。
* **退款承诺：** 弹窗底部必须常驻 `🛡️ 7天无理由退款` 标识。

## 5. 数据安全与防泄漏规则 (Security Rules)

**开发侧必须严格执行以下后端过滤逻辑：**

1. **判定规则：** 遍历交易记录。
* `IF` (标的当前持仓数量 > 0): 标记为 `Encrypted`。
* `ELSE`: 标记为 `Public`。


2. **后端脱敏：**
* 对于 `Encrypted` 的行，后端 API **绝对不能** 返回真实的 `symbol_code` 和 `exact_price`。
* 必须在服务器端替换为 `Sector_Name` (行业名) 和 `null`，防止前端源码泄露。


3. **时间模糊化：**
* 加密行的时间戳仅返回到 **分钟级** 或 **小时级**，前端格式化为范围 (e.g., "09:30-10:00")。

## 6. 总结 (Summary)

此页面的 UI 设计 (`image_bc0879.png`) 完美执行了 **“中心化诱导布局”**。

* **视觉焦点：** 用户的视线会自然落在 **黄色标签** -> **锁图标** -> **解锁按钮** -> **红色浮盈** 这一条黄金线上。
* **商业闭环：** 通过展示“公开行”建立能力信任，通过展示“加密行的浮盈”制造贪婪，最终通过“按钮”完成收割。
